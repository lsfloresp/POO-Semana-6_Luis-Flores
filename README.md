## Objetivos

### Objetivo general
Aplicar los principios de la Programación Orientada a Objetos mediante el desarrollo de un programa en Python, utilizando clases base y clases derivadas, con una adecuada organización del código.

### Objetivos específicos
- Implementar herencia entre clases para reutilizar atributos y métodos.
- Aplicar encapsulación en la definición de atributos del sistema.
- Definir correctamente métodos y atributos en las clases.
- Crear instancias de las clases y comprobar su funcionamiento.

---

## Descripción del proyecto
El presente proyecto corresponde a la práctica de la Semana 6 de la asignatura Programación Orientada a Objetos. El programa fue desarrollado en el lenguaje Python utilizando el entorno de desarrollo PyCharm.

El sistema implementa un modelo orientado a objetos para representar dispositivos electrónicos, permitiendo aplicar conceptos fundamentales como herencia y encapsulación de forma estructurada.

---

## Conceptos de Programación Orientada a Objetos aplicados
- Clases y objetos
- Herencia
- Encapsulación
- Métodos
- Reutilización de código

---

## Funcionamiento del programa
Se define una clase base denominada `Dispositivo`, la cual contiene atributos y métodos comunes a todos los dispositivos electrónicos, como la marca y el estado de encendido.

A partir de esta clase se derivan las clases `Televisor` y `Radio`, las cuales heredan las características de la clase base e implementan comportamientos específicos mediante la sobrescritura de métodos.

En el archivo principal del programa se crean instancias de las clases y se ejecutan los métodos correspondientes para verificar el correcto funcionamiento del sistema.

---

## Ejecución del programa
El programa se ejecuta a través del archivo `main.py` desde el entorno de desarrollo PyCharm.

---

## Resultado esperado
Al ejecutar el programa, se muestra en consola el estado de cada dispositivo creado, indicando su tipo, marca y estado de encendido.

---

## Conclusión
El desarrollo de esta práctica permite afianzar los conceptos de Programación Orientada a Objetos en Python, facilitando la comprensión de la herencia y la encapsulación mediante una implementación modular y estructurada.
